Fuerza MRA - Simulación de tocar la malla

Para lograr crear la simulación de tocar la malla, se puede utilizar un control PD que suaviza el movimiento del objeto del sistema MRA, o bien, se puede usar un sistema MRA críticamente amortiguado, ambos aplicados unicamente durante el periodo de tiempo que se está tocando. Para ésta simulaciòn se utilizó la segunda opción (sistema MRA críticamente amortiguado).


Donde cada gráfica representa: Línea morada: Sistema MRA subamortiguado, con un T = 3.0s.
  • Se le aplica una fuerza en el tiempo 0.0, y se estabiliza en el tiempo 3.0
 Línea verde: Sistema MRA subamortiguado, con un T = 3.0s.
  • Se le aplica una fuerza en el tiempo 0.0, y se estabiliza en el tiempo 3.0
  • Se le quita la fuerza en el tiempo 5.0, y se estabiliza en el tiempo 8.0
 Línea azul: Sistema MRA críticamente amortiguado, con un T = 1.0s.
  • Se le aplica una fuerza en el tiempo 0.0, y se estabiliza en el tiempo 1.0
  • Se le quita la fuerza en el tiempo 5.0 (ahora con un T = 3.0s para volver a un sistema MRA subamortiguado), y se estabiliza en el tiempo 8.0. (Logrando un comportamiento igual al de la línea verde)
 Línea amarilla: Sistema MRA críticamente amortiguado, con un T = 1.0s.
  • Se le aplica una fuerza en el tiempo 0.0, y se estabiliza en el tiempo 1.0
  • Se modifica la fuerza (reduciéndola a la mitad) en el tiempo 2.0, y se estabiliza en el tiempo 3.0
  • Se le quita la fuerza en el tiempo 5.0 (ahora con un T = 3.0s para volver a un sistema MRA subamortiguado), y se estabiliza en el tiempo 8.0.

Para esta simulaciòn se establecieron los siguientes valores constantes: k: 0.2 (Kg/s^2) m: 0.01 (Kg) b: 0.02667 (Kg/s) d_t: 1.0/30.0 (s)

La especificaciòn de la fuerza aplicada se hace a travès de un mouse 3D:
  • Traslaciòn Z: Aplica una fuerza proporcional al movimiento del mouse 3D: TZ*0.2, TZ ∈ [-1.0, 1.0]
